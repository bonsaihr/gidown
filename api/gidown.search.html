gidown.search¶

This module implements functions and classes for acquiring images from google image search.

license:Apache2, see LICENSE for more details.
class gidown.search.GoogleSearchImage(google_json_dict: dict)[source]¶

Bases: object

Class used to store information about images from google image search. Two GoogleSearchImage are considered the same if they have the same image URL.

download(download_all=False, check_ext=True) → None[source]¶

Download the image and store raw bytes into self.image. Image type (extension) is automatically detected from the raw bytes if possible (type).

Parameters:
  • check_ext – Checks if extensions is correct. Raises ValueError if extension cannot be determined from image.
  • download_all – Also download thumbnail and store it in self.thumbnail.
download_thumbnail() → None[source]¶

Download the thumbnail and store raw bytes into self.thumbnail. Image type (extension) is automatically detected from the raw bytes if possible (thumbnail_type).

save(filename: str, auto_ext=False)[source]¶

Save image to filesystem. If the image was not downloaded using download it is done now.

Image type (extension) is automatically detected from the raw bytes if possible, otherwise the type is determined from the google image search dictionary.

Parameters:
  • filename – Path to store the image to.
  • auto_ext – Automatically add the file extension.
save_thumbnail(filename, auto_ext=False)[source]¶

Save thumbnail to filesystem. If the thumbnail was not downloaded using download_thumbnail it is done now.

Image type (extension) is automatically detected from the raw bytes if possible, otherwise the type is determined from the google image search dictionary.

Parameters:
  • filename – Path to store the thumbnail to.
  • auto_ext – Automatically add the file extension.
gidown.search.image_query(query: str, *restrictions: gidown.advanced.QuerySetting, autocorrect=False) → typing.List[gidown.search.GoogleSearchImage][source]¶

Query google for images that satisfy all the restrictions.

Parameters:
  • query – Query string, supports all advanced google search options.(see gis.query for wrappers around often used options)
  • restrictions – Variable amount of advanced search options. (see gis.advanced)
  • autocorrect – Auto-correct queries that have a high likelihood of a misspell (e.q. “color grenn”)
Returns:

List of unique GoogleSearchImage objects that can be downloaded and saved.
